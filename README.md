# Energy Catch Game ⚡

Energy Catch Game adalah game interaktif berbasis Computer Vision
yang menggunakan kamera untuk mendeteksi jari tangan pemain.

Pemain harus menyentuh bola energi di layar menggunakan jari telunjuk
untuk mendapatkan skor sebanyak mungkin dalam waktu 30 detik.

## Teknologi
- Python
- OpenCV
- MediaPipe

## Cara Menjalankan
1. Clone Repository Ini
git clone https://github.com/bintangcloud/energy-catch-game.git
cd energy-catch-game

2. Install dependencies
Gunakan perintah berikut untuk menginstal semua dependencies:
pip install opencv-python mediapipe

3. Jalankan program
python EnergyCatchGame.py

## Kontrol
- Gerakkan jari telunjuk untuk menyentuh bola
- Tekan **R** untuk restart
- Tekan **Q** untuk keluar
